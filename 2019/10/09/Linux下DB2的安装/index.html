Linux下DB2的安装

发表于2019-10-09|更新于2019-10-09|数据库
|字数总计:311|阅读时长:1分钟|阅读量:

前言

一只在银行工作的程序猿，工作需要使用DB2数据库比较多，于是在自己买的服务器上安装DB2来学习DB2相关命令的使用。这里记录一下在Linux下安装DB2的方法。

Setp1:下载DB2 Linux安装包

进入IBM官网下载DB2安装包。点击下载

Step2：通过FTP上传至服务器并解压tar包

tar -zxvf <tar包>
得到一个server_dec文件夹,执行server_dec下的db2_install开始安装

分别键入YES YES 和SERVER走流程安装DB2

Step3：创建DB2用户及组

useradd db2adm1
useradd db2inst1

Step4：创建db2管理服务器实例

  • root用户进入 /home/db2adm1
  • 执行 /opt/ibm/db2/V11.5/instance/dascrt -u db2adm1

Step5：创建DB2数据库实例

  • root用户进入 cd /home/db2inst1
  • 执行 /opt/IBM/db2/V8.1/instance/db2icrt -u db2inst1 db2inst1

Step6：检查db2服务端口,没有则添加(一般安装程序已经自动添加):

vi /etc/services

Step7：设置db2实例运行环境变量

  • 进入db2inst1用户: su - db2inst1
  • 启动数据库: db2start
    如果出现以下错误切换root用户,安装缺少组件。yum install numactl
  • 设置允许db2通过tcpip方式访问: db2set DB2COMM=tcpip
