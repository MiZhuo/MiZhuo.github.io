Vue的安装及项目搭建

null简介

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式JavaScript框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，方便与第三方库或既有项目整合。

在这个大数据时代，对于互联网项目性能越来越高，前后端分离是大势所趋，虽说前后端只需要专攻自己的领域，但是作为一名经验丰富(熟练掌握CTRL+C/CTRL+V等复杂技术)的老程序猿，除了要学习后端技术的同时，也要对前端技术进行了解。这样才会有一个充满希望的未来！而Vue.js可以说是目前国内最流行的前端框架了，它是以数据驱动和组件化的思想构建的。相比于Angular.js，Vue.js提供了更加简洁、更易于理解的API，使得我们能够快速地上手并使用Vue.js。

Vue的教程十分详细，大家也可以移步查看官网的教程。 👉官网教程 👈

nullStep1(安装Vue.js)

注意：Vue 不支持 IE8 及以下版本。

  • 使用NPM安装,npm install vue,NPM 能很好地和诸如 webpack 或 Browserify 模块打包器配合使用。使用NPM推荐使用国内镜像,速度更快,除了使用NPM.

  • 直接引入<script>标签,<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>,Vue会被注册为一个全局变量。

  • 使用官方命令行工具 (CLI),为单页面应用 (SPA) 快速搭建繁杂的脚手架。
    我这里使用npm进行安装:

  • 全局安装vue-cli工具

    cnpm install vue-cli -g

nullStep2(新建Vue项目)

  • vue init webpack (你的项目名称)

  • 安装中不用全选

  • 安装完成后，就可以运行项目了

  • 打开浏览器，进入http://localhost:8080/就可以看到我们的项目了

nullStep3(目录介绍)

主要的文件都是放在src中

  • 在src/assets中新建css、img、js文件夹
  • src/components中存放自己的vue组件
  • src/router存放路由配置信息
  • src/lib存放模拟的数据
  • 新建src/store存放vuex
  • 新建src/views存放路由组件
