前言

本文仅作为新手参考教程,前端使用Vue+ElementUi搭建,后端使用SpringBoot+Mybatis提供数据接口。

脚手架的安装这里就不讲解了,可以浏览前面的博客。希望对你有帮助!

前端项目环境搭建

  • 安装Element-UI
    • 安装命令:npm install element-ui -s
    • 引入组件: import ElementUI from 'element-ui'
    • 引入样式Css:import 'element-ui/lib/theme-chalk/index.css'
    • 使用组件:Vue.use(ElementUI)
  • 安装axios
    • 安装命令:npm install axios -s
    • 引入组件:import axios from 'axios'
    • 使用组件:Vue.prototype.$axios = axios
  • 安装Vue的状态管理Vuex
    • 安装命令:npm install vuex -s
    • 引入组件:import Vuex from 'vuex'
    • 使用组件:Vue.use(Vuex)
  • 安装日期处理类库moment.js
    • 安装命令:npm install moment -s
    • 引入组件:import moment from 'moment'
  • 安装NProgress进度条
    • 安装命令:npm install nprogress -s
    • 引入组件:import NProgress from 'nprogress'
    • 引入样式css:import 'nprogress/nprogress.css'
    • 使用组件:NProgress.start(); NProgress.done();
